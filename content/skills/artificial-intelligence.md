Here's how I've used AI:

- Created custom prompts for things like my homelab, resume reviews, technical writer assistant, and more.
- Created a Python script that prompted users to fill in the required fields for the homepage template.
- Running multiple LLMs locally through [LM Studio](https://lmstudio.ai/) (mostly from the [Qwen3 family](https://huggingface.co/collections/Qwen/qwen3)).
- Replaced GitHub Copilot with local LLMs in VS Code using the [Cline extension](https://cline.bot/).
- Developed my portfolio website using Google Bard/Gemini, Claude, GitHub Copilot, and local LLMs.
- Locally hosted [SearXNG anonymous search engine](https://docs.searxng.org/) in a Docker container and added it as an MCP server so local LLMs can access the internet.
