Here's how I've used AI:

- Created custom prompts for things like my homelab, resume reviews, technical writer assistant, and more.
- Running multiple LLMs locally through [LM Studio](https://lmstudio.ai/) (mostly from the Qwen3 family).
- Replaced GitHub Copilot with local LLMs in VS Code using Cline.
- Developed my website using Google Bard/Gemini, Claude, GitHub Copilot, and local LLMs.
- Set up locally hosted SearXNG anonymous search engine in a Docker container and added it as an MCP server so local LLMs can access the internet.
